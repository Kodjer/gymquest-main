# 🎯 Улучшения системы квестов

## 📅 Дата обновления

3 января 2026

## ✅ Выполненные улучшения

### 1. 💾 Резервная копия

- Создана резервная копия оригинального банка квестов: `questBank.backup.ts`
- Возможность восстановить исходную версию при необходимости

### 2. 🎨 Визуальные демонстрации

Добавлена поддержка визуальных инструкций для упражнений:

- **Типы демонстраций**: изображения, GIF анимации, видео, YouTube
- **Иконки-подсказки**: эмодзи для быстрой визуальной идентификации типа упражнения
- **Визуальный интерфейс**: красивые карточки с градиентами для демонстраций

### 3. 📝 Пошаговые инструкции

Добавлены детальные пошаговые инструкции (stepByStep):

- **Пошаговая техника выполнения** с эмодзи-нумерацией
- **Подробное описание каждого шага**
- **Визуально выделенные блоки** для легкого восприятия

### 4. 📊 Расширение банка квестов

Значительно увеличено количество квестов без добавления новых упражнений:

#### Силовые тренировки (Strength)

**Легкий уровень (Easy):**

- Отжимания новичка (оригинал)
- Отжимания новичка плюс (8 повторений)
- Отжимания с длинными подходами (4х6)
- Приседания для начинающих (оригинал)
- Приседания с паузой (3х8 с паузой)
- Приседания с увеличенным объемом (4х12)
- Медленные приседания (темп 5-5)
- Планка на старте (оригинал)
- Планка с увеличением времени (3х30 сек)
- Планка с дополнительными подходами (4х25 сек)
- Динамическая планка (с переходами)
- Легкие выпады (оригинал)
- Выпады с увеличенным объемом (3х10)
- Шагающие выпады (3х16)

**Средний уровень (Medium):**

- Отжимания среднего уровня (оригинал)
- Отжимания 5x10
- Отжимания с узкой постановкой (акцент на трицепс)
- Отжимания с широкой постановкой (акцент на грудь)
- Отжимания с дефицитом (увеличенная амплитуда)
- Темповые отжимания (3-1-1)
- Приседания с весом (оригинал)
- Приседания 5x12
- Приседания с паузой внизу (3 сек)
- Темповые приседания (3-0-1)
- Приседания сумо (широкая постановка)
- Приседания на одной ноге с ассистом
- Планка с усилением (оригинал)
- Планка 5x40
- Планка с подъемом ног
- Боковая планка (по 30 сек на сторону)
- Планка с касанием плеч
- Планка-паук (с подтягиванием колена)

#### Прогрессия квестов

Квесты усложняются за счет:

- 📈 **Увеличения подходов**: 3 → 4 → 5 подходов
- 🔢 **Увеличения повторений**: 5 → 8 → 10 → 12 → 15 → 20
- ⏱️ **Увеличения времени**: 20 → 25 → 30 → 40 → 45 → 90 секунд
- 🐌 **Изменения темпа**: обычный → медленный → темповой → с паузой
- 📐 **Изменения постановки**: узкая → обычная → широкая
- 💪 **Добавления веса**: без веса → с легким весом → с тяжелым весом
- 🔄 **Добавления вариаций**: статические → динамические

### 5. 🎯 Структура новых квестов

Каждый квест теперь содержит:

```typescript
{
  title: "Название упражнения",
  instructions: "Краткая инструкция",
  description: "Подробное описание",
  tip: "Совет для выполнения",
  category: "strength" | "cardio" | "flexibility" | "wellness",
  difficulty: "easy" | "medium" | "hard",
  baseXP: number,
  visualDemo: {
    type: 'gif' | 'image' | 'video' | 'youtube',
    url: string,
    thumbnail: '🏋️‍♂️' // эмодзи-иконка
  },
  stepByStep: [
    "1️⃣ Первый шаг",
    "2️⃣ Второй шаг",
    "3️⃣ Третий шаг",
    // и так далее
  ]
}
```
